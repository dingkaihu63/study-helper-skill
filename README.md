> 别慌。把教材交给 AI，剩下的交给我们。**study-helper-skill** 是一套专为 **Claude Code / OpenClaw / OpenHands** 等顶级 AI Agent 打造的“工程级”复习自动化工具链。它能让你的 AI 助手瞬间化身为“拥有 10 年教学经验的命题组老教授”，直接为你产出能直接打印进考场的专业复习指南。

-----

## 🔥 为什么要开发这个 Skill？

在 Agent 时代，简单的“总结一下”已经过时了。本项目解决了 AI 处理学术资料时的三大“死穴”：

| 痛点 | 传统 AI 总结 | **study-helper-skill** |
| :--- | :--- | :--- |
| **上下文断层** | 遇到整本教材直接报错或截断 | **内置智能分块引擎，实现全书深度扫描** |
| **交互深度** | 敷衍的“摘要”，不理会你的实际进度 | **交互式诊断：根据你的剩余时间和薄弱点量身定制** |
| **结果落地** | 凌乱的终端文本，无法复用 | **工业级 `.docx` 产出，带目录、公式与专业排版** |

-----

## 🛠️ 三位一体的自动化链路

本 Skill 不仅仅是一段 Prompt，它是一个由三个高性能组件构成的**自动化工厂**：

  * **📂 结构化预处理 (`文件预处理.py`)**：支持 PDF/Word/PPT 全格式识别。它会自动清洗冗余信息，并按章节逻辑切分数据，确保 AI 拿到的每一行字都是“干货”。
  * **🧠 专家级大脑 (`analyze_summarize.md`)**：注入了 10 年经验的命题逻辑。它会自动建立“前置知识 -\> 核心定理 -\> 典型避坑指南”的深度逻辑网。
  * **📄 文档重构器 (`资料文件生成.py`)**：基于 `python-docx` 的专业排版引擎。一键产出符合学术标准的 Word 文档，**拿去打印店就能直接印。**

## 🚀 30 秒开启“学霸模式”

1.  **环境准备**：`pip install -r requirements.txt`
2.  **执行指令** (以 Claude Code 为例)：
    ```bash
    claude skill run study-helper-skill --materials_path="./控制工程基础.pdf" --course_name="自动控制"
    ```
3.  **收割成果**：稍等片刻，在项目根目录下，一份排版精美的 `[课程名]_复习指南.docx` 将自动静候。
